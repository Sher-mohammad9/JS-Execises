Difference between Function Declaration and function expression
Function Declaration Function Expression
Pahal function keyword likha ja ta hai. use ke bade ek unique name likh ke function ko declare kiya jata hai. Pahal variable declare kiya jata hai. use ke bade function keyword ka use karke anonymous function variable main assing kiya jata hai.
Function declaration ki hoisting hoti hai. Function expression ki hoisting nhi hoti hai.
Function declaration ko as a variable trite nhi kiya ja sakta. Function experssion ko as a variable trite kiya ja sakta hai.
Function declaration first class function nhi hote hai. Function experssion first class function hote hai.


Difference between Function expression and arrow function
Function Expression Arrow Function
Pahal variable declare kiya jata hai. use ke bade function keyword ka use karke anonymous function variable main assing kiya jata hai. Pahal variable declare kiya jata hai. use ke bade bina function keyword ka use kiy anonymous function variable main assing kiya jata hai.
Function expression ka syntax beda hota hai. Arrow ka syntax chota hota hai.
Function experssion me single line statement likh ne ke liye pura syntax use hota hai. Arrow function me single line statement likh ne ke liye {Curly braces} mandatory nhi hai.
const experssion = function (str) { console.log("Wecode Academy") }; const arrow = (str) => console.log("Wecode Academy");
